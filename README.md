# Python List Programs 🐍

This repository contains basic Python programs for learning list operations.

## Topics Covered

- Create list
- Add elements (append, insert)
- Remove elements (remove, pop)
- Update list values
- Find elements (search, last item)
- Count elements
- Length of list
- Max and Min
- Sort list (asc/desc)
- Reverse list
- Copy list
- Join lists
- Clear list
